# Grammar

## 1. 一般现在时（Simple Present Tense）

一般现在时是理解其他时态的基础，主要用法：

* 表示现在的想法、情感和状态
* 表示习惯性的、反复性的动作或者事情
* 表示一般概念、俗语、不可否认的客观真理
* 有时也表示将来

### 1.1 例句

```
She walks to the house.
He usually sleeps at school.
```

### 1.2 Must have known

* 一般现在时用于陈述句中
* 一般现在时中，第三人称单数的动词后面加 s 或者 es
* if 和 when 引导的状语从句用一般现在时
